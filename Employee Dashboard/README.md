# Srashtikasoftware Employee Dashboard

An interactive employee dashboard built with Python and Streamlit for Srashtikasoftware company.

## Features

- Dashboard Overview with key metrics
- Employee Performance tracking
- Department Analytics
- Attendance Tracker
- Project Management
- Resource Allocation

## Installation

1. Clone or download this repository
2. Install the required packages:
   ```
   pip install -r requirements.txt
   ```

## Running the Dashboard

### Option 1: Using the start_app script (Recommended)
```
python start_app.py
```

### Option 2: Direct Streamlit command
```
streamlit run dashboard_app.py
```

## Dashboard Pages

1. **Dashboard Overview**: Key metrics and summary charts
2. **Employee Performance**: Performance analysis and top performers
3. **Department Analytics**: Department-wise comparisons
4. **Attendance Tracker**: Attendance monitoring and analysis
5. **Project Management**: Project status and progress tracking
6. **Resource Allocation**: Resource distribution and management

## Requirements

- Python 3.7 or higher
- Streamlit
- Pandas
- NumPy
- Plotly

## Customization

You can customize the dashboard by modifying `dashboard_app.py`:
- Add new metrics and visualizations
- Modify existing charts
- Add new pages
- Change styling in the CSS section
